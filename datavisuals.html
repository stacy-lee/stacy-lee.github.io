Demographic Clustering Sample

I performed supervised learning on a random sample of Ameren customer demographics, representing about 0.1% of Ameren's customers. These were created for practice. In the plots below, customer political affiliation is labeled as 'R' for Republican, 'I' for Independent, 'N' for Non-registered, and 'D' for Democrat.

The pairwise plots below show how a customer's political affiliation is represented by a customer's age, estimated home value, census median home value, and home built year. Evidently, there are no noticeable patterns in this sample of data.

Some further analysis of this data sample continues to lack obvious patterns. Below is a principal component score analysis of home built year with customer age, which does not have clear patterns to describe the data since the points lie relatively evenly across the positive and negative principal component scores. Cluster classification using 3 clusters, Class 0, Class 1, and Class 2, was also utilized to find patterns, but the overall accuracy was quite low at about 27%.
